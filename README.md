# Реализация генерации изображения фрактального пламени, основанного на Chaos Game.

## Функциональные требования
* Реализация цветного алгоритма генерации фрактального пламени;
* Возможность запуска как в однопоточном, так и в многопоточном режиме (регулируется константой GoroutineCount);
* Гибкая настройка генерации: размер изображения, набор трансформационных функций;
* Поддержка следующих трансформаций: Handkerchief, Disc, Heart, Diamond, Swirl;
* Поддержка логарифмическая гамма-коррекция;
* Поддержка параметра симметрии.

## Входные, выходные данные
### Входные данные
* Размеры изображения (ширина и высота);
* Количество итераций для генерации фрактала;
* Список трансформационных функций и их параметры.

### Выходные данные
Изображение фрактального пламени (image.png)

## Инструкция по запуску
Выполнить: `go run .\cmd\run\main.go`
